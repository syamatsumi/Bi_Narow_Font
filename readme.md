# Biはんぶんゴシック/Biはんぶんこ明朝

ユニバーサルデザインフォントの
[BIZ UDゴシック](https://github.com/googlefonts/morisawa-biz-ud-gothic)
[BIZ UD明朝](https://github.com/googlefonts/morisawa-biz-ud-mincho)
を50％に長体化しただけのフォントです。

# Biわりびきゴシック/Biわりびき明朝
が75％で

# Biほそめゴシック/Biほそめ明朝
が90％です。


## 特徴

縮めただけです。Excelとか長体化ができないソフトで使うつもりで作りました。


### 余談

そもそも和文コンデンスドフォント（和文ナローフォント、和文長体フォント）は、
プロ向けのフォントしか見当たらないから会社で使えなくて困るんよ。

Excelでコンデンスドフォントを使うと書式の設定がクッソめんどくさいので
「これいっこ指定すれば文字詰め放題！」
って状態にしたくって作りました。

ビルドはエラーだらけだし処理はアホほど重い極めて雑なことしてるし、
高さの揃え方とか幅の詰め方とか、まぁまぁ雑な事をしているので、
破綻してるグリフも多数あるだろうから教えて貰えるとうれしいな……


# ライセンス

縮めただけなのでBIZ UDゴシック、BIZ UD明朝のライセンスに従ってください。
